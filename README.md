# 📌 README.md — Soomin’s Website  
> **v1(Main) → v2(Last) 대규모 기능 & UI 업데이트**

이 프로젝트는 개인 포트폴리오 웹사이트로,  
HTML/CSS 기반의 정적 페이지에서 시작하여  
현재 버전에서는 **동적 기능 + 테마 시스템 + 모바일 UI 강화**까지 적용되며 크게 발전했습니다.

아래는 업데이트된 내용에 대한 요약입니다.

---

# 🚀 업데이트 요약

현재 버전은 단순 정적 웹페이지 → “상호작용 많은 포트폴리오 사이트”로 발전한 버전입니다.

## ✔ 1. **JavaScript 기능 추가**
현재 버전에는 새로운 `script.js` 파일이 추가되었으며  
웹사이트의 인터랙션 전반을 담당합니다.  
(예: 테마 변경, 모달, 메뉴 토글, 카테고리 필터링 등)

주요 기능:
- ⏱ **시간대별 인사 자동 변경**  
- 📱 **모바일 메뉴 토글(햄버거 메뉴)**  
- 🎨 **다크/라이트 테마 전환 + LocalStorage 저장**  
- 🗂 **포트폴리오 카테고리 필터**  
- 🔍 **프로젝트 클릭 시 상세 모달 표시**  
- ⬆ **위로가기 버튼 클릭 시 메뉴 active 상태 초기화**

---

## ✔ 2. **테마 시스템 추가 (Dark/Light 모드)**
현재 CSS에서는 테마 기능이 새로 추가됨.

### 추가된 내용
- `:root[data-theme="dark"]`, `:root[data-theme="light"]` CSS 정의  
- Primary / Accent / Text 색상 테마 별도 세팅  
- `theme__toggle` 버튼 추가

---

## ✔ 3. **모바일 메뉴 토글(햄버거 버튼) 추가**
처음에는 **PC 전용 메뉴**만 있었지만,  
현재는 **모바일 UI 완전 대응**이 이루어졌어.

### 주요 변화
- `<button class="header__toggle"><i class="fa-solid fa-bars"></i></button>` 추가
- 모바일에서 메뉴 자동으로 열고/닫기 가능

---

## ✔ 4. **포트폴리오 기능 강화**
현재는:
- 카테고리 버튼에 class 추가  
- 프로젝트에도 타입 class 부여  
- JS로 개수 자동 계산 후 badge 갱신  
- 필터링 애니메이션 적용
